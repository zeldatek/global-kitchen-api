# The Global Kitchen API

## Tech Stack

- Runtime: Node.js
- Framework: Express.js
- Database: MongoDB Atlas
- ODM: Mongoose
- Configuration: dotenv

## Features

- Create recipes
- Retrieve recipes
- Filter recipes by category
- Update recipes
- Delete recipes
- MongoDB schema validation
- Global error handling
- 3-Tier architecture
